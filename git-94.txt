Branches
========

git branch

git branch -d release1  (to delete a branch)

git checkout -b release2  ( to create and checkout branch)

git branch -d release1  (to delete branch)

git push origin release1  ( to upload branch into central repo)
or
git push path/of/repo release1

open a file unser .git/config
ex:-
[branch "release1"]
	remote = origin
	merge = refs/heads/release1
[branch "release2"]
	remote = origin
	merge = refs/heads/release2
Then it will work git push command from release branches.

git push origin -d release1  (to delete branch in central repo)
